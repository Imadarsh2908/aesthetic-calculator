# 🧮 Cyberpunk Web Calculator

A sleek, modern, and fully responsive web-based calculator featuring a stunning **Neon Cyberpunk Aesthetic** and glassmorphism design. Built with pure HTML, CSS, and JavaScript.

## ✨ Features
- **Modern Neon Aesthetic:** Features a dynamic animated gradient background that transitions through deep space black, neon cyan, and vibrant magenta.
- **Glassmorphism UI:** The main calculator interface uses a semi-transparent dark backdrop with blur effects, soft rounded corners, and a glowing cyan shadow to provide a premium feel.
- **Interactive Grid Animation:** An animated matrix of squares in the background lights up with an intense cyan glow as you interact with the app.
- **Fully Responsive:** Adapts seamlessly to different screen sizes.
- **No Dependencies:** Built entirely with vanilla web technologies. No frameworks or build tools required.

## 🚀 How to Run

Because this is a vanilla web project, you don't need `npm`, `Node.js`, or any complex build setup! You can run it in a few simple ways:

**Method 1: Direct Open (Easiest)**
1. Clone or download this project.
2. Double-click on `index.html` to open it directly in your default web browser.

**Method 2: VS Code Live Server**
1. Open the project folder in Visual Studio Code.
2. Install the **Live Server** extension.
3. Right-click `index.html` and select **"Open with Live Server"**.

**Method 3: Terminal Local Server**
Open your terminal in the project directory and run one of the following:
- Using Python: `python -m http.server`
- Using Node.js: `npx http-server`
Then, open the provided localhost URL (e.g., `http://localhost:8000`) in your browser.

## 🛠️ Tech Stack
- **HTML5** for structure
- **Vanilla CSS3** for the Cyberpunk styling, animations, and responsive design
- **Vanilla JavaScript** for calculator logic and operations
- **FontAwesome** for button icons
- **Google Fonts** (Quicksand) for modern typography
